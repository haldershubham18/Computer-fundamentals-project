# Static Portfolio Webpage  

A clean, semantic, and responsive personal portfolio website built using HTML and CSS.  
This project demonstrates modern web development practices, proper version control, and professional documentation.

## Features  
- Semantic HTML5 structure  
- Responsive CSS layout  
- Modern, clean design  
- Easy to customize  

## Installation & Usage  
- 1.Download the zip 
- 2.Open index.html in your browser.
- 3.Customize content in index.html and style.css as needed.

## Technology Used
- HTML5
- CSS3
- Git & GitHub
- VS Code
